Desenvolver uma plataforma web full-stack para registro, classificação automática e acompanhamento de incidentes de segurança cibernética, com painel administrativo, relatórios e modelo de Machine Learning retreinável.
Objetivos Específicos
01
Classificação Automática via ML

02
Controle de Acesso por Papel (RBAC)

03
Dashboard com KPIs e Visualizações
Gráficos em tempo real, filtros avançados e busca de texto completo
04
Relatórios PDF e Exportação CSV
Com filtros aplicados, quebra de linha automática e modo admin global
05
Auditoria Completa de Alterações
Histórico de cada mudança de status, nota e categoria com timestamp
