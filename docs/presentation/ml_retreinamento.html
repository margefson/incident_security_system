O que muda no retreinamento
Novas categorias adicionadas pelo admin são incorporadas automaticamente
Incidentes reais do banco aumentam a base de treinamento com dados de produção
Acurácia recalculada e exibida imediatamente no painel após o retreinamento
Data de atualização registrada no metrics.json e exibida no painel ML
Detalhes Técnicos
Endpoint: POST /retrain em Flask :5001
Payload: JSON com array de {text, category}
Modelo: salvo em classifier.pkl via joblib
Fallback: Node.js PDFKit quando Flask indisponível
Benefícios
Modelo aprende com dados reais de produção continuamente
Sem necessidade de re-deploy — retreinamento via interface web
Rastreabilidade: data e acurácia de cada versão do modelo
